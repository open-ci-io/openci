# OpenCI ランナー へようこそ
[![Deploy GitHub Apps](https://github.com/open-ci-io/openci/actions/workflows/github-apps-cd.yml/badge.svg)](https://github.com/open-ci-io/openci/actions/workflows/github-apps-cd.yml)

OpenCIランナーは、GitHub Actionsの、より高速で低価格なランナーを提供するサービスです。

OSSで開発しておりますので、ご意見・ご要望がありましたら気軽にissueを作成、もしくは[Slackチャンネル](https://join.slack.com/t/open-ci-io/shared_invite/zt-3ectydtjn-GbcRVXkX~DKVFZNAqB~XtQ)でご連絡ください。

## 料金
2025年12月31日までは無料です。ぜひご意見・ご要望をお聞かせください！

## ランナースペック
テスト期間中は、一律v8CPU, 16GB RAM, 240GBのランナーを提供します。
